# Breast Cancer Detection Using Infrared Thermography

## 📌 Overview
This project presents a computer-aided breast cancer detection system using infrared thermography. The system analyzes thermogram images to classify them as Normal or Suspicious.

## 🚀 Features
- Non-invasive and radiation-free screening
- Image preprocessing and segmentation
- Feature extraction and symmetry analysis
- Classification of thermogram images
- Flask-based web interface for image upload and prediction

## 🛠 Technologies Used
- Python
- OpenCV
- NumPy
- Flask
- Machine Learning / Image Processing

## 🖥 How It Works
1. User uploads thermogram image.
2. Image is preprocessed and segmented.
3. Features are extracted.
4. System classifies the image.
5. Result is displayed on web interface.

## 🎯 Objective
To provide an accessible and user-friendly tool for early breast cancer screening, especially in low-resource regions.
